Amir Ethernet Continuity Tester

An Arduino-based cable tester for continuity checks, miswire detection, and short detection.

Developed by Amir Mobasheraghdam

Features

  • 8-channel cable testing
  • Serial diagnostic output
  • Short circuit detection
  • Miswire detection

Platform

  • Arduino Uno and Nano support
  • Compatible with RJ45 breakout boards
  • PlatformIO support
  • Clean project structure for GitHub
