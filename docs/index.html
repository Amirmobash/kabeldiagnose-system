Amir Ethernet Cable Tester

A simple Arduino-based tool for testing Ethernet cable continuity, wiring order, and short circuits.

Created by Amir Mobasheraghdam

What It Checks

  • Continuity across all 8 Ethernet wires
  • Incorrect wire order
  • Short circuits between wires
  • Clear serial monitor test results

Supported Hardware

  • Arduino Uno
  • Arduino Nano
  • RJ45 breakout boards
  • PlatformIO or Arduino IDE
